# 💻 プログラミング言語アンケートアプリ

## 概要
G'sACADEMYの受講生を対象にしたプログラミング言語に関するアンケートアプリです。
回答を収集・表示する仕組みをPHPとMySQLで実装しました。

Ver2では、Chart.JSをしてリアルタイムで集計しています。その後はＤＢからデータを取得してＰｙｔｈｏｎで集計予定です。
なお、公開用にホスト名等はLocalにてアップしています。

## 使用技術
- PHP
- MySQL
- HTML / CSS
- XAMPP（ローカル開発環境）
- さくらインターネット（本番サーバー）
- FileZilla（FTPアップロード）
- GitHub（ソースコード管理）

## 機能
- アンケートフォームの表示
- 回答のDBへの登録
- 回答一覧の表示

## アンケート項目
- ニックネーム
- 好きなプログラミング言語
- 経験年数
- お金になると思う言語
- 転職に有利な言語
- 学習コストが低い言語
- 使っている領域
- 次に学びたい言語
- 現在のプログラミング使用状況
- 各項目の理由・コメント

## セキュリティ対策
- XSS対策：htmlspecialcharsによる入力値のエスケープ
- SQLインジェクション対策：PDOプリペアドステートメントの使用

## ファイル構成
- `index.php` → アンケートフォーム画面
- `register.php` → 回答のDB保存処理
- `result.php` → 回答一覧表示画面
